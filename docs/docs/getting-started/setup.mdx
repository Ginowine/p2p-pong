# Setting Up Pear

First, let's install the Pear runtime on your system.

## Install Pear

Run this command in your terminal to install Pear globally:

```bash
npm install -g pear
```

Once installed, you can verify the installation by running:
```bash
pear --version
```

This should output the version number of Pear you installed.
