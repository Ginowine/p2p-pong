# UI Shell: HTML & CSS

First, create the `ui/` directory that holds all frontend files:

```bash
mkdir ui
```

---

## `ui/index.html`

The HTML is intentionally minimal. It defines two "screens" (lobby and game) using `display: none` / `display: flex` toggling — no router library needed.

```html

  <head>

  </body>
</html>
```

**Key structural points:**

- Both screens always exist in the DOM — `app.js` switches between them by toggling `display`.
- The `canvas` element is fixed at 800x480 so the physics can use absolute pixel coordinates.
- The overlay (win/disconnect screen) is `position: absolute` over the canvas, blurred with `backdrop-filter`.
