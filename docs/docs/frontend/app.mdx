# Frontend: app.js

The frontend is the UI layer that runs in Chromium! It handles:
- User input
- Game state management
- Communication with the backend
- Drawing via the renderer

## Install dependencies and run

Now that all files are in place, install the dependencies and launch the app:

```bash
npm install
pear run -d .
```

The `-d` flag runs in development mode with live reload. To test multiplayer on a single machine, open a second terminal and run the same command — both instances can connect to each other.
